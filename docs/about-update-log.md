## 一、关于本站

`THArchive` 是一个面向 **东方 Project 同人接力活动** 的归档网站。

这个站点的目标不是做一个普通博客，也不是做一个只靠搜索引擎偶然搜到的零散信息页，而是尽量把分散在不同平台、不同时间、不同主办方手中的活动信息，整理成一个 **可以查找、可以回看、可以继续补充** 的归档系统。

目前本站主要围绕以下几件事展开：

- 展示已经收录的接力活动
- 为活动提供独立详情页
- 支持按时间、状态等方式浏览活动
- 提供前台投稿入口，让活动主办方自行补充资料
- 由站点维护者进行审核、整理与发布

你可以把它理解成：

> 一个为东方同人接力活动服务的资料站、索引站和归档站。

---

## 二、本站目前能做什么

目前已经上线或基本可用的功能包括：

- **活动列表浏览**
  - 可以查看已收录活动
  - 支持列表视图与日历视图

- **活动独立页面**
  - 每个活动会有自己的详情页
  - 目前会展示标题、简介、活动说明、日期、角色、主办方、封面图、图集和归档链接等内容

- **活动投稿**
  - 活动主办方可以通过前台表单提交活动资料
  - 投稿进入待审核状态，不会直接公开

- **后台审核与补充**
  - 管理员会对投稿进行核对、整理、补图和补标签
  - 审核通过后活动会正式发布到前台

- **轮播与首页展示**
  - 首页或专题区域可以展示近期活动或指定年份活动

---

## 三、适合谁使用

这个网站主要面向以下几类访客：

- **活动主办方**
  - 想补录自己主办过的活动
  - 想为活动留下一个更稳定、更容易查找的归档入口

- **普通参与者 / 观众**
  - 想查找某个角色、某个年份、某类活动
  - 想回看已经结束的活动资料

- **整理者 / 档案爱好者**
  - 想把零散活动信息尽量标准化
  - 想帮助补全旧活动资料

## 四、新人引导

如果你是第一次来到这里，可以按下面的顺序使用本站。

### 1. 如果你只是想看活动

建议先去：

- `活动列表`
- `日历视图`
- `首页轮播 / 推荐区`

你可以先快速浏览，再进入单个活动页查看详细内容。

### 2. 如果你是活动主办方，想补录活动

建议准备这些资料后再来投稿：

- 活动标题
- 一句话简介
- 东方角色
- 活动日期
- 主办方名称
- 活动说明
- 总结专栏或主归档链接
- 一张封面图

如果你还有这些内容，也可以一并提供：

- 独立归档站链接
- 宣传图 / 海报 / 图集
- 原始活动文本
- 其它备注

### 3. 投稿之后会发生什么

投稿成功后，活动不会立刻公开，而是会进入待审核状态。

管理员通常会做这些事情：

- 检查标题、日期、主办方、角色等基础信息
- 补充封面图或图集
- 整理归档链接
- 补充后台分类与标签
- 核对信息后发布

这意味着：

- 你前台提交的内容不需要一次填得过于复杂
- 一些更细的标签、分类和补充字段会由管理员后续整理

### 4. 如果你发现信息不完整怎么办

如果某个活动页面里：

- 缺图
- 缺链接
- 信息有误
- 简介或说明不够完整

可以通过【】联系站点维护者，或等待后续补录。

## 五、关于收录范围

当前本站主要收录与 **东方 Project 同人接力活动** 相关的内容。

优先考虑的对象包括：

- 角色日 / 角色主题接力
- 节日 / 纪念日接力
- 主题向接力企划
- 具有明确主办方与活动时间的接力活动

以下内容是否收录，可能会视情况决定：

- 资料过少、缺乏基本信息的活动
- 难以确认来源的二次转述
- 与“接力活动”关联较弱的其它企划

如果你不确定某个活动是否适合投稿，可以先提交，后续由管理员判断。

## 七、使用说明

### 投稿建议

为了提高审核效率，投稿时建议尽量做到：

- 标题写清楚活动名
- 日期尽量使用准确日期
- 说明里写明活动形式和基本规则
- 至少提供一个可访问的主归档链接
- 封面图尽量使用活动海报、主视觉或宣传图

### 浏览建议

如果你想快速找到活动，可以优先根据：

- 年份
- 日期
- 活动状态
- 角色
- 主办方

来检索或筛选。
